毛玻璃蒙版

毛玻璃蒙版，本质上讲是两层图，底层图清晰完整，顶层图模糊残缺，并且两者位移完全一致，因此叠加到一起后像是一张图。为了确保位移一致，我们用了 background : fixed (这个是效果的关键点)，他能使背景图片相对于浏览器进行固定。
